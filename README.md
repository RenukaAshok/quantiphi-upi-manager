# UPI Money Manager

A Streamlit application that mimics a banking transaction dashboard.

## Features
- Automated keyword-based categorization
- Income and expense calculation
- Category-wise spending analytics
- Interactive category selector
- Cashback reward estimation
- Transaction timeline feed

## Categories
- Food & Dining
- Travel
- Salary
- Miscellaneous

## Tech Stack
- Python
- Streamlit
- Pandas

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
````
